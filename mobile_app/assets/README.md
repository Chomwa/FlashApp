# Flash App Assets

This directory contains app icons and images for the Flash payment app.

## Required Files for Expo

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Current Status

Placeholder files needed. The app will work without these but will show default Expo icons.

## Creating Icons

You can:
1. Use online tools like https://www.appicon.co/
2. Design custom icons with the Flash lightning theme
3. Use the default Expo icons for development

## Flash App Branding

- **Primary Color**: #2196F3 (Blue)
- **Theme**: Lightning/Flash payment
- **Icon**: Lightning bolt or flash symbol
- **Style**: Modern, clean, minimal
